# Smart Timetable Generator

## 📌 Overview
This project generates an optimized timetable for students by efficiently allocating subjects, time slots, and constraints.

## 🚀 Features
- Automatic timetable generation
- Conflict-free scheduling
- Custom subject allocation

## 🛠️ Tools Used
- Python
- Excel (Dataset)

## 📊 Output
The system generates a structured and balanced timetable.
